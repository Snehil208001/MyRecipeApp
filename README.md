# MyRecipeApp 🍽️

MyRecipeApp is a modern Android application built with **Jetpack Compose**, **Retrofit**, and **Coil** to provide a seamless recipe browsing experience.

## 📌 Features
- 📝 **Browse Recipes** - View a list of delicious recipes.
- 🔍 **Search Recipes** - Find recipes based on ingredients or cuisine.
- 📷 **Load Images Efficiently** - Uses Coil for image loading.
- 🌐 **Fetch Data from API** - Retrofit is used to retrieve recipes dynamically.

## 🛠️ Tech Stack
- **Kotlin** - Primary programming language.
- **Jetpack Compose** - Modern UI toolkit.
- **Retrofit** - API calls for fetching recipes.
- **Coil** - Efficient image loading.
- **ViewModel** - Manages UI-related data.
- **Material 3** - UI components for a better user experience.
