# MyRecipeApp 🍽️

MyRecipeApp is a modern Android application built with **Jetpack Compose**, **Retrofit**, and **Coil** to provide a seamless recipe browsing experience.

## 📌 Features
- 📝 **Browse Recipes** - View a list of delicious recipes.
- 🔍 **Search Recipes** - Find recipes based on ingredients or cuisine.
- 📷 **Load Images Efficiently** - Uses Coil for image loading.
- 🌐 **Fetch Data from API** - Retrofit is used to retrieve recipes dynamically.
- 🔄 **Coroutines for Async Calls** - Smooth data fetching using Kotlin Coroutines.
- ⚡ **Error Handling** - Implements try-catch-finally for robust network operations.

## 🛠️ Tech Stack
- **Kotlin** - Primary programming language.
- **Jetpack Compose** - Modern UI toolkit.
- **Retrofit** - API calls for fetching recipes.
- **Coil** - Efficient image loading.
- **ViewModel** - Manages UI-related data.
- **Coroutines** - Handles background network requests.
- **Material 3** - UI components for a better user experience.

## 📚 Key Concepts Implemented  
This project is based on the following concepts:
1. **JSON & Data Classes** - JSON structure and mapping to Kotlin data classes.
2. **TheMealDB API** - Used for fetching recipe data.
3. **ViewModel & LiveData** - Manages UI state efficiently.
4. **Retrofit & Coroutines** - Network calls with async execution.
5. **Try-Catch-Finally** - Error handling in API requests.
6. **Jetpack Compose UI** - Building UI dynamically with state management.
